# ai-appointment-scheduler
An AI-powered appointment scheduling workflow built with n8n that validates booking requests, checks Google Calendar availability, creates appointments automatically, logs bookings, and sends confirmation emails.

Features:

AI receptionist
Calendar availability checking
Google Calendar integration
Automatic booking
Google Sheets logging
Confirmation emails
AI decision making

Tech Stack:

n8n
OpenAI GPT-4.1 Mini
Google Calendar
Google Sheets
Gmail

Workflow:

Customer Request
↓
AI Receptionist
↓
Calendar Specialist
↓
Check Availability
↓
Create Event
↓
Log Booking
↓
Send Confirmation
